在棒糖如何写博客

2021-11-10 by hanling

以下说明仅供棒糖科技内部员工参考, 用来引导团队成员配置个人环境以及编写发布博文

环境配置

1. 安装 nodejs

参考: nodejs.org

2. 克隆项目

$ git clone git@[bongmi-gitlab-host]:bongmi-ios/blog.git

3. 初始化博客的运行环境

$ sh manager.sh ensure_env

4. 开通权限

将 github 账号提供给 hanling@bongmi.com 并接受邀请

编辑与发布

1. 新建一篇博文

$ sh manager.sh new [file_name]

2. 编辑博文

$ sh manager.sh edit [file_name]

3. 本地实时预览博客

$ sh manager.sh run

4. 发布你写的文章

$ sh manager deploy

操作行为规范

1. 在 gitlab 上统一使用 master 分支, 使用 rebase 进行多人开发, 而不是 pull | merge
2. 禁止中文命名文件, 文章名统一按照小写单词加中划线的规范命名
3. 图片统一放在 source/images 路径下, 多张图按文章名维护子文件夹
4. 使用 sh manager.sh run 实时在本地预览你的文章, 确保发布的文章排版正常
5. 如需开启评论, 在文章尾部附上”评论脚本”
